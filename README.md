# Identificação e Mapeamento Participativo de Conflitos Ambientais em Formosa - GO

## Resumo

  A proposta deste projeto de pesquisa e inovação é coletar, organizar, problematizar e analisar dados de
conflitos ambientais que ocorrem no município de Formosa, produzindo uma base eletrônica de dados desses
conflitos. O Mapa de Conflitos Ambientais do município de Formosa-GO será desenvolvido a partir da localização
e georeferenciamento de áreas de conflitos socioambientais, por intermédio de tecnologia GPS (Global Positioning
System). Os dados georeferenciados serão indicados em um mapa virtual, que poderá ser disponibilizado na rede
mundial de computadores e acessado por aplicativo mobile (em smartphones ou tablets) ou via interface web, em
computadores convencionais.
